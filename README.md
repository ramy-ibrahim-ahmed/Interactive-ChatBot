مرحباً! بخصوص شروط استخدام الكميات المجانية الإضافية في المبيعات، إليك أبرز النقاط المتوفرة:\n\n- يجب أولاً تفعيل خيار \"استخدام الكميات المجانية في المبيعات\" من الإعدادات.\n- عند التفعيل، يظهر حقل خاص لإدخال الكمية المجانية الإضافية عند إنشاء فاتورة المبيعات.\n- لا يجوز إدخال كمية مجانية تتجاوز الكمية المحددة مسبقاً في شاشة العروض الترويجية.\n- إذا كان لدى العميل عرض ترويجي، تستطيع أيضاً إدخال كميات مجانية يدوياً بشرط تفعيل المتغير المتعلق بذلك.\n- يتم احتساب الضريبة على الكمية المجانية فقط إذا كان خيار احتساب الضريبة مفعلاً لهذه الكميات.\n\nلاحظ أن المعلومات ليست شاملة لكل الشروط التجارية أو الاستثناءات، ولم توضح الوثائق مثلاً الفترات الزمنية أو الفئات المحددة بشكل دقيق. إذا رغبت في تفاصيل أكثر أو واجهت حالة خاصة، يسعدني التصعيد لسpecialist الدعم لمتابعة استفسارك!
